# Demo Notes

- Demo video: https://www.youtube.com/watch?v=2IbIenrjvfg
- Website: https://www.agentdrydock.com/

This demo illustrates the execution-first loop:

1) Run in a disposable container
2) Capture real failure signals (compiler/test output)
3) Propose a patch
4) Re-run to verify
5) Return only verified outcomes

> Note: Demo content may include randomly generated code. No proprietary or production code is included.
